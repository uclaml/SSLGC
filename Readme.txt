This package contains the Matlab implementations of GPA, OLLGC, and SSLGC, as well as the datasets used in our kdd13 submission [1].


Please run the script files 'test_Coauthor.m', 'test_cora.m',  'test_IMDB.m' and 'test_PubMed' to reproduce the experimental results. 


[1] Quanquan Gu, Charu Aggarwal, Jialu Liu and Jiawei Han, Selective Sampling on Graphs for Classification,
  KDD, 2013.
